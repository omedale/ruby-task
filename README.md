# Installation
`
$ git clone https://github.com/omedale/ruby-task.git
`

`
$ cd ruby-task
`

`
$ bundle Install
`
## Start app
`
$ ruby lib/app.rb
`

## Test
`
$ bundle exec rspec
`

## Ruby Version 

`
  ruby 2.2.1p85
`

## Assumptions
- Assuming data in 'data/campaigns.json' represents local campaigns
